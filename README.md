# **Integrated Fullstack & AWS DevOps Program**
**Program Duration:** 12 Weeks (balanced between development and DevOps skills)
**Program Focus:** Build full-stack applications with Python and master AWS cloud infrastructure, containerization, and CI/CD pipelines.

## **Course Modules**
### **1. Python Fullstack Development (6 Weeks)**
* Django & FastAPI Framework for backend development
* REST APIs & Microservices architecture
* Database design & optimization (SQL and NoSQL basics)
* AI/ML Fundamentals (basic integration into applications)
* Frontend essentials (React/JavaScript basics to integrate with backend APIs)

### **2. AWS Cloud & DevOps Fundamentals (6 Weeks)**
* Introduction to AWS Cloud Services (EC2, S3, Lambda, RDS, IAM)
* Docker & Containerization (build, manage containers)
* CI/CD pipelines using Jenkins and GitLab
* Infrastructure as Code (IaC) basics with AWS CloudFormation or Terraform
* Monitoring & Logging on AWS

## **Learning Outcomes**
* Develop and deploy robust full-stack Python applications
* Understand AWS core services and deploy applications securely on the cloud
* Build containerized applications using Docker
* Automate application deployment using CI/CD pipelines
* Manage infrastructure as code for scalable and maintainable deployments
